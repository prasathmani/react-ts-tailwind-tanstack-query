# Vitejs + React + TypeScript + Tailwind CSS + Tanstack Query Boilerplate

## This repository is 🔋 battery packed with:

- ⚡️ Vite js
- ⚛️ React 18
- ✨ TypeScript
- 💨 Tailwind CSS 4
- 🦀 Tanstack Query
- 🌐 Axios

### Setup Project

```
npm i
```

### Run local server

```
npm start
```

### Build Application

```
npm run build
```

### ENV

Please include a .env in the root folder in below format

```
VITE_API_BASE_URL='http://localhost/api'
```
